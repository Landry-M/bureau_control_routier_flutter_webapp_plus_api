⚠️  ACTION IMPORTANTE (Optionnelle mais recommandée) :

Pour générer des PDF de haute qualité, installez wkhtmltopdf :

  macOS :
  $ brew install wkhtmltopdf

  Ubuntu/Debian :
  $ sudo apt-get install wkhtmltopdf

Sans wkhtmltopdf, le système créera des fichiers HTML qui peuvent
être imprimés en PDF manuellement depuis un navigateur.
